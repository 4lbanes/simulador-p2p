o trabalho é fazer um simulador de um sistema P2P, onde constrói a estrutura e fazer algumas buscas nela 

O arquivo de entrada(JSON ou TXT) deve conter a estrutura definida, o sistema monta em memória, pode ter uma interface bacana 

No arquivo deve conter:
- Quantos nós 
- Quantidade máxima e mínima de vizinho de cada nó
- Recursos para cada nó
- Arestas(quem é vizinho de quem)

O simulador além de simular, tem que mostrar o rastro do trajeto (rastro visual ou textual)

Permitir fazer buscas, o nó que começa a busca, o recurso que busca e escolher o algoritmo 

também deve mostrar um log final contendo: 
- achou ou não
- ⁠onde achou 
- ⁠quantas mensagens foram trocadas 
- ⁠quantos nós foram visitados
